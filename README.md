# App Gallery

A curated collection of useful Android apps.  
Tested on older hardware, but compatibility may vary. Includes FOSS tools, patched apps, web wrappers, and community submissions.

---

## 📦 What's Inside

This gallery is a mix of app types:

- **Open Source Apps** – Fully FOSS replacements for system tools.
- **Patched Apps** – Popular apps modified to remove ads and trackers (ReVanced, microG).
- **Web-to-App Wrappers** – Lightweight web apps packaged for Android (including my own creations).
- **Trusted Proprietary Apps** – Useful apps from trusted developers that respect user privacy.

---

## 🧩 How to Add an App

You can contribute to this gallery in two ways:

1. **Submit your own creation** – If you built an Android app, web wrapper, or tool that also runs on older devices, you can submit it to the gallery.
2. **Suggest an app** – If you found a great app that isn't listed here, open an issue with the details.

New submissions will appear at the top of the gallery with a **"NEW"** badge for 3 days, giving them a temporary spotlight.

---

## 🎯 Purpose

This is not a generic app store.  
It is a curated toolkit for people who want to:

- Keep older Android phones running smoothly.
- Reduce bloatware and tracking.
- Discover lightweight, useful apps that actually work.

---

## 🛠️ Built With

- HTML, CSS, and Vanilla JavaScript
- Hosted on GitHub Pages
- Data stored in a single `data.json` file
